## 📁 Файлы проекта

### `.pcf` — конфигурация выводов (Pin Configuration File)

Файл с расширением `.pcf` используется в ESP-IDF для назначения имён пинов (GPIO) и их функций.

Пример содержимого:
GPIO2 LED
GPIO4 RELAY


📌 **Как использовать:**
- Поместите файл в корень проекта
- В коде можно обращаться к пинам по имени, а не по номеру
- Упрощает перенос проекта на другую плату

---

### `.pzw` — проект PICSimLab

Файл `.pzw` — это проект симуляции в **PICSimLab**. Он сохраняет:
- Схему подключения
- Добавленные компоненты (LED, реле, датчики)
- Настройки симуляции
- Положение элементов на рабочем поле

📌 **Как открыть:**
1. Установите [PICSimLab](https://picsimlab.org)
2. Откройте программу → **File → Open Project**
3. Выберите файл `*.pzw`

---

### `.pcf.ppa` — вспомогательный файл PICSimLab

Создаётся автоматически при работе с проектом `.pzw`. Хранит дополнительные настройки компонентов. **Не требует ручного редактирования.**
